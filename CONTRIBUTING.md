# راهنمای مشارکت در Raven Downloader

<div dir="rtl">

## 🎯 روش‌های مشارکت

از مشارکت شما استقبال می‌کنیم! می‌توانید به این روش‌ها کمک کنید:

### 🐛 گزارش باگ
در [Issues](https://github.com/YOUR-USERNAME/raven-downloader/issues) یک Issue جدید بسازید:
- شرح دقیق مشکل
- لینکی که استفاده کردید (در صورت امکان)
- لاگ خطا (error log)

### 💡 پیشنهاد ویژگی
ایده‌های جدید را در بخش Discussions مطرح کنید.

### 🔧 مشارکت در کد
1. مخزن را Fork کنید
2. یک Branch جدید بسازید
3. تغییرات را Commit کنید
4. Pull Request بفرستید
